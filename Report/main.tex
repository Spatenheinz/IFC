This project presents a simple imperative programming language, with a built in assertion language, which makes it possible to formally prove correctness of programs, based on Floyd-Hoare Logic and predicate transformer semantics.
The outcome of this project is an application for both dynamically evaluating programs and statically verifying them. The static verification works similarly to tools such as Why3, which we compare our solution to.
